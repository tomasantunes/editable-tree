# editable-tree
Editable Tree

An utility to fill out nested text fields with an extra panel for URLs.

![ko-fi](https://www.ko-fi.com/img/githubbutton_sm.svg)](https://ko-fi.com/Y8Y2M1UI)

## Testing

npm install

npm start

## Building
